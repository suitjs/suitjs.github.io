# SuitJS Website    
Official site repository.

## Install
* Clone/Pull the repository.
* Make sure to have NodeJS and NPM.
* `npm run init`
* Make sure your Git user is setup and stored.
* `gulp serve`
* **[Localhost](http://localhost:8080)**

## Publish
* `gulp publish`
